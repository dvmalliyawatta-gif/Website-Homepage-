# Website-Homepage

This repository contains a responsive homepage UI implementation based on the provided design reference. The goal of this task is to demonstrate clean front-end development practices, responsive layout structuring, and maintainable code organization using modern web technologies.

Project Overview
Assignment: Homepage UI Coding
Type: Static Frontend (HTML, CSS, JavaScript)
Framework: Bootstrap 5
Fonts: Playfair Display, Lato (Google Fonts)
Icons: Bootstrap Icons

File Structure
kngu/
├── index.html        # Main HTML file (all page sections)
├── css/
│   └── style.css     # Custom styles built on top of Bootstrap 5
├── js/
│   └── main.js       # Custom interactivity (filter tabs, newsletter, announcement bar)
└── README.md         # Project documentation

How to Run Locally
No build tools or installations needed. Just open the file directly in your browser:
Download and unzip the project folder
Open index.html in any modern browser (Chrome, Firefox, Edge, Safari)
All assets load from CDN — an internet connection is required for fonts, Bootstrap, and placeholder images

Live Demo - Hosted on vercel
https://website-homepage-1o2zwsgsv-d-malliyawatta-s-projects.vercel.app

Notes
All navigation links use href="#" as this is a static UI demo, not a full application
Images are loaded from Unsplash CDN URLs — no local image files are required
The filter tab system uses data-category attributes on product cards for clean, maintainable JS logic
